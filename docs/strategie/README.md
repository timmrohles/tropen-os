# Strategie-Dokumente

Hier liegen Strategie-relevante Dokumente, die nicht in `docs/product/`
oder `docs/synthese/` passen — typischerweise:

- Status-Berichte für externes Sparring
- Marken-Strategie-Snapshots
- Strategie-Reflexionen ohne ADR-Reife

## Bestand

- `status-YYYY-MM-DD.md` — periodische Status-Berichte für Sparring/Retrospektive
