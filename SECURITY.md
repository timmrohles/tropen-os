# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest (main) | ✅ |

## Reporting a Vulnerability

**Bitte keine öffentlichen GitHub Issues für Sicherheitslücken.**

Sicherheitslücken meldest du direkt per E-Mail an:
**hello@tropen.de**

Bitte include in deiner Meldung:
- Beschreibung der Lücke
- Schritte zur Reproduktion
- Mögliche Auswirkungen

## Reaktionszeiten

| Schweregrad | Reaktionszeit |
|-------------|---------------|
| Kritisch (CVSS ≥ 9.0) | < 24 Stunden |
| Hoch (CVSS 7.0–8.9) | < 7 Tage |
| Mittel / Niedrig | < 30 Tage |

Du erhältst eine Bestätigung innerhalb von 48 Stunden.
Wir informieren dich über den Fortschritt der Behebung.

## Scope

Betroffen: tropen-os.de und alle Subdomains
Nicht betroffen: Third-party Services (Supabase, Vercel, Dify)
